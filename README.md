# PushFromYq

```text 

listening to all YUQUE update information from PushBear push 

YUEQUE => WECHAT

```

- PushFromYq是什么
    - 对接语雀api进行各种更新的push
    - push 目标当然是wechat
    - 使用的push工具为[【server酱】](http://sc.ftqq.com/3.version)
